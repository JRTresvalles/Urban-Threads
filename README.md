# Urban Threads
A conceptual fashion eCommerce website created as a portfolio project to explore modern UI design, layout systems, and scalable frontend development.

> ⚠️ This is a conceptual/portfolio project and not a production-ready website.

## About the Project
Urban Threads focuses on delivering a clean and modern shopping experience inspired by contemporary fashion brands. This project covers both the design and development process—from initial wireframing to final UI implementation.

I was responsible for designing the interface, creating wireframes, and developing the frontend using a component-based approach.

## Features
- Modern eCommerce-inspired UI
- Component-based structure
- Responsive design across devices
- Product-focused layouts and sections
- Simulated user interactions (e.g., cart, product views)

## Tech Stack
- HTML / SCSS / JavaScript
- Gulp (task runner)
- Sass (CSS preprocessor)

## Purpose
- Practice end-to-end product design (wireframing → UI → development)
- Showcase frontend architecture and component structuring
- Demonstrate scalable styling using Sass and Gulp workflow

## Disclaimer
This project is for demonstration purposes only and does not include real transactions, authentication, or backend functionality.
